LL is a form of a sequential collection and it does not have to be in order.
A LL is made up of independent nodes that may contain any type of data and each node has a reference to the next node in the link.

Array vs LL:
LL is not a built-in data struture
LL is not indexed
LL is not contigious

Types of LL:
Single LL
Circular LL
Doubly LL
Circular Doubly LL
